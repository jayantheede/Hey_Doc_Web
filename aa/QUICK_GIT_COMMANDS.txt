QUICK GIT COMMANDS TO PUSH YOUR CODE

1. Open a NEW Command Prompt or PowerShell (after restarting computer)
2. Navigate to your project folder:
   cd C:\Users\Jayanth\Desktop\aa

3. Run these commands one by one:

git init
git add .
git commit -m "Initial commit - Aastha Homoeo Clinic Hospital Management System"
git remote add origin https://github.com/jayantheede/Hey_Doc_Web.git
git branch -M main
git push -u origin main

That's it! Your code will be pushed to GitHub.

If you get authentication errors, you may need to:
1. Sign in to GitHub in your browser
2. Create a Personal Access Token
3. Use the token as your password when prompted

Your repository will be available at:
https://github.com/jayantheede/Hey_Doc_Web
